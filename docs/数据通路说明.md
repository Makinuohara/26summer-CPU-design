# 单周期数据通路说明

第一版 CPU 每个时钟周期完成一条指令。

```text
PC
  -> instr_mem
  -> control_unit / imm_gen / regfile
  -> ALU
  -> data_mem
  -> writeback mux
  -> regfile
```

## 关键控制信号

| 信号 | 作用 |
| --- | --- |
| `reg_write` | 允许写回寄存器 |
| `mem_read` | 允许读数据存储器 |
| `mem_write` | 允许写数据存储器 |
| `mem_to_reg` | 写回数据来自内存而不是 ALU |
| `alu_src` | ALU 第二操作数来自立即数 |
| `branch` | 当前指令为条件分支 |
| `jump` | 当前指令为 `jal` |
| `alu_op` | 交给 `alu_control` 进一步译码 |

## PC 更新

```text
jump                  -> PC + imm
branch && take_branch -> PC + imm
otherwise             -> PC + 4
```
